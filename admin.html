🔥 Firebase Echtzeit-Sync

Daten zwischen Geräten synchronisieren — kein Datenverlust beim Gerätewechsel.

⚫ Nicht konfiguriert
Kostenloses Konto erstellen auf console.firebase.google.com → Neues Projekt → Realtime Database → Konfiguration kopieren.
Oder Firebase-Config als JSON einfügen:
📋 Empfohlene Firebase Sicherheitsregeln 
{
  "rules": {
    "praxen": {
      "$praxisId": {
        ".read":  "auth != null || true",
        ".write": "auth != null || true"
      }
    }
  }
}
⚠️ Für Produktion: Authentifizierung aktivieren und Regeln verschärfen.
💡 Ohne Firebase: Daten nur auf diesem Gerät. Mit Firebase: Sync zwischen allen Geräten (PC, Tablet, Handy) in Echtzeit. Kostenlos bis 1 GB Speicher & 10 GB/Monat Traffic.
